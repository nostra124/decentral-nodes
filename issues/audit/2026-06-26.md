## Correctness gap found (install)

Walking the install surface surfaced a **high-severity** bug: `make
install` ships no verbs for the `-node` commands — the rename never
reached `PACKAGES`/`COMMANDS`/`share/`, and seven nodes aren't registered
at all. Every installed node except forgejo/webmin/usermin is broken.
Filed **BUG-058** (3.3.4). The unit suites missed it because they only
exercise the dev tree and assert an empty staged dir — a testing blind
spot, addressed by the items below.
