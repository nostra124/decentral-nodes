# Audit 2026-05-13

"every release" cadence in `skills/audit.md` §2 — nine releases
(1.0.1, 1.1.0, 1.2.0, 1.3.0, 1.4.0, 1.5.0, 1.5.1, 1.5.2, 1.6.0,

## Findings

### Forward gaps (§3.1: behaviour exists, no `done/` file)

1. **Dispatcher surface** — `help`, `version`, `modules` predate
   the issue tree. **→ file FEAT-027** documenting the dispatcher
   contract.
2. **`bip13` libexec plugin** — base58 encode/decode commands;
   used by every BIP-32 round-trip in the codebase. No origin
   feature. **→ file FEAT-028.**
3. **`bip32` plugin** — the BIP-32 implementation itself.
   BUG-013/014 fixed *defects* in it, but the plugin's existence
   has no `done/` origin file. **→ file FEAT-029.**
4. **`bip39` plugin** — words / create / check / seed
   subcommands. **→ file FEAT-030.**
5. **`daemon` plugin** — start / stop / monitor / space verbs.
   No tests, no feature trace. Unclear what this is for. **→ file
   BUG-015** ("daemon plugin is undocumented and untested") so a
   maintainer can decide: document, remove, or scope a real feature.
6. **`wif` plugin** — encode / decode (in repo) and `-u` flag
   (referenced in vectors, **not** implemented). **→ file FEAT-031**
   for the plugin's origin contract; the missing `-u` flag stays
   on FEAT-025's roster.
7. **`command:bip49-create` / `command:bip84-create`** — dispatcher
   functions that call a never-defined `command:bip32-create`. The
   wrappers are dead code today. **→ file BUG-016** ("bip49-create
   / bip84-create dispatcher entries call undefined function").

### Reverse gaps (§3.2: `done/` describes missing behaviour)

None identified. Every shipped FEAT-/BUG- description maps to

## Hardening proposal (separate audit follow-up)

Three bugs (BUG-013, BUG-014, and the open BUG-016) have all been
"function name diverged from its definition; bash doesn't catch
it." A lint step that greps each plugin for `command:<name>`
invocations and asserts each `<name>` is defined-as-function in
the same file would catch this defect class proactively. Filing
this as **FEAT-032** ("lint: undefined-command-call detector").
